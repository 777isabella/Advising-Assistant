# Advising-Assistant
The application will function as an academic advising assistant intended to support both 
faculty and students. Users will be required to register a username and password. Faculty 
users will have access to student academic records, including transcripts, degree plans, and 
projected schedules of remaining courses, including prerequisites. Student users will have 
access to their own academic information, allowing them to monitor their progress, review 
completed courses, and view upcoming course requirements. Both faculty and student 
users will be able to generate summaries of completed coursework, as well as 
recommendations for courses to take in the future. Course names will serve as links that 
provide detailed information regarding course descriptions and prerequisites. We want to 
further investigate and compare which programming languages would best suit our project. 
We will first analyze the needs of our program and what tools and languages can best fit 
these needs. We plan to begin with tackling the functionality of our program and when 
functionality is nearly complete, we will begin implementing design features to make it 
more user-friendly and enjoyable
